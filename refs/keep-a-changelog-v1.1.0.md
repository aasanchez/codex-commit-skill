# Keep a Changelog v1.1.0 (Summary)

## Section headers

Use only the sections that apply:

- `### Added`
- `### Changed`
- `### Deprecated`
- `### Removed`
- `### Fixed`
- `### Security`

## Bullet rules

- Use unordered bullets (`-`).
- Keep bullets user-meaningful; avoid file lists.
- Keep each bullet short and clear.
